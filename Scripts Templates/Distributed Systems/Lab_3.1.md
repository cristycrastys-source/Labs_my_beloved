## Теоретические основы

### gRPC
Современный высокопроизводительный фреймворк RPC. Использует Protocol Buffers для сериализации данных и HTTP/2 в качестве транспортного протокола.

### RabbitMQ
Брокер сообщений, реализующий протокол AMQP 0-9-1. Работает по принципу: Producer → Exchange → Queue → Consumer.

### Docker
Платформа для разработки, доставки и запуска приложений в контейнерах.
