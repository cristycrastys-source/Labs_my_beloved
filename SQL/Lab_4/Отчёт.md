## Вывод

В ходе выполнения лабораторной работы были освоены:

**Оконные функции PostgreSQL:**
- `RANK()`, `DENSE_RANK()`, `ROW_NUMBER()` — ранжирование строк внутри окон
- `LAG()`, `LEAD()` — доступ к предыдущим и следующим строкам
- `SUM()`, `AVG()`, `COUNT()` в качестве оконных функций

**Ключевые конструкции:**
- `PARTITION BY` — разделение данных на группы (окна)
- `ORDER BY` в окне — определение порядка строк
- `ROWS BETWEEN ... AND ...` — оконный фрейм для точного задания границ

**Практические навыки:**
- Вычисление нарастающих итогов (Running Total)
- Расчет скользящих средних (Moving Average)
- Ранжирование данных внутри групп
- Сравнение строк с соседними (Lead/Lag анализ)

**Статистика выполнения:**
- Часть 1 (общие задания): 3 запроса
- Часть 2 (индивидуальные задания): 3 запроса
- Всего запросов: 6

Все запросы выполнены корректно, оконные функции применены согласно требованиям, группировка `GROUP BY` не использовалась там, где требуется сохранение всех строк результата.
