# Самостоятельная работа 1: Работа с внешними приложениями

**Цель работы:** Научиться подключаться к базе данных PostgreSQL с использованием Python (библиотека `psycopg2`), создавать структуры данных (схемы и таблицы), наполнять их данными, выполнять базовые SQL-запросы и визуализировать результаты в Yandex DataLens.

---

## Инструменты

- Python 3.8.10
- Jupyter Notebook
- Библиотека `psycopg2-binary`
- PostgreSQL 16.13 (Docker)
- Yandex DataLens
