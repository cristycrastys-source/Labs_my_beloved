## Часть 3. CRUD операции (Локально)

### Задание 1.3*. Спецпроект — customers_nyc

**Задание:** Создать копию клиентов из NYC, удалить с индексом 10014, добавить колонку event, заполнить 'thank-you party'.

**Выполнение:** Последовательно выполняются операции:
1. `CREATE TABLE ... AS SELECT` — создание таблицы-копии с данными клиентов из Нью-Йорка
2. `DELETE` — удаление клиентов с почтовым индексом 10014
3. `ALTER TABLE ... ADD COLUMN` — добавление текстовой колонки `event`
4. `UPDATE` — заполнение колонки значением 'thank-you party'

### Задание 2.3*. CRUD — failed_emails

**Задание:** Создать таблицу failed_emails из bounced писем, добавить retry='N', удалить тему 'Welcome'.

**Выполнение:**
1. `CREATE TABLE failed_emails AS SELECT * FROM emails WHERE bounced IN ('t', 'Yes')` — создание таблицы с отскочившими письмами
2. `ALTER TABLE failed_emails ADD COLUMN retry VARCHAR(1)` — добавление колонки для отслеживания повторных попыток
3. `UPDATE failed_emails SET retry = 'N'` — установка значения 'N' (нет повторной попытки)
4. `DELETE FROM failed_emails WHERE subject = 'Welcome'` — удаление писем с темой 'Welcome'
